# machi-list

`machi-list` は、市町村と店ジャンルを掛け合わせた一覧ページを軸に、全国展開を目指す地域店舗検索サイトです。

MVPでは愛知県を対象に、ネットカフェ、ゲームセンター、アダルトショップ、カラオケ、サウナの5ジャンルを扱います。地図は補助機能として位置感、ルート、所要時間の確認に使い、検索流入ページでは一覧、比較、予約、クーポン、通販、周辺情報への導線を重視します。

## 起動

```bash
npm start
```

URL:

```text
http://127.0.0.1:4173/
```

## 確認

```bash
npm run check
```

## 方針

- URL構造は `prefecture / city / ward / genre / shop` を基本にする
- 一覧ページは `都道府県 > 市区町村 > ジャンル` の検索意図に合わせる
- 店舗データはGoogle Place IDなど外部IDを持てる形にする
- 予約、クーポン、通販、駐車場、周辺飲食店への導線をページ種別ごとに分ける
- 地図は主役ではなく、位置確認とルート確認の補助機能にする
